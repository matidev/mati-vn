# Setup Environment

```
# 
# Install Ruby and Jekyll
$ make setup

# Update bundle
$ make update

# Update JS dependencies
$ make js-update

# Run Jekyll in local by using this command
$ make
```

# Dev Notes

- Update parameters on `_config.yml` file.
- Also for `_data` folder.
- Run Jekyll in local by using this command

```
$ make
```

- Open browser, enter the url http://localhost:4000/
